HTML中属于布尔类型的属性

  1. noshade,用来表示有无阴影，多用于在<hr />标签当中;
  2. ckecked,用来表示是否默认选中，多用于单选按钮<input type="radio" />和多选按钮<input type="checkbox" />中，需要注意的是，在<input type="radio" />中需要用相同的name才能达到单选框的效果;
  3. selected,用来表示是否优先显示，用于下拉表单<select></select>和只能下拉表单<datalist>中;
  4. autofocus,用来自动获取焦点，用于有输入框的<input />标签中;
  5. readonly,只读属性，可以获取焦点，数据会传递给后台,适合于能输入的文本框，比如text
  6. disabled,禁用属性，可以使其无法获取焦点，数据无法传递给后台,在输入文本和按钮都可以用
  7. require,用来表示必须选择，多用于<input />标签中

HTML编码规范

  1. class 必须单词全字母小写，单词间以 - 分隔
  2. class 必须代表相应模块或部件的内容或功能，不得以样式信息进行命名。
  3. 元素 id 必须保证页面唯一。
  4. 同一页面，应避免使用相同的 name 与 id。
  5. id 建议单词全字母小写，单词间以 _ 分隔。同项目必须保持风格一致。
  6. 对于无需自闭合的标签，不允许自闭合。常见无需自闭合标签有 input、br、img等。
  7. 标签使用必须符合标签嵌套规则 div不得置于p中，tbody必须置于table中
  8. 布尔类型的属性，建议不添加属性值。
  9. 自定义属性建议以 xxx- 为前缀，推荐使用 data-。
  10. 使用 HTML5 的 doctype 来启用标准模式，建议使用大写的 DOCTYPE。
  11. 在html标签上设置正确的 lang 属性。
  12. 指定字符编码的 meta 必须是 head 的第一个直接子元素。
  13. 页面必须包含 title 标签声明标题。
  14. title 必须作为 head 的直接子元素，并紧随 charset 声明之后。
  15. 保证 favicon 可访问。
  16. 引入 CSS 时必须指明 rel="stylesheet"。
  17. 引入 CSS 和 JavaScript 时无须指明 type 属性。
  18. JavaScript 应当放在页面末尾，或采用异步加载。
  19. 禁止 img 的 src 取值为空。延迟加载的图片也要增加默认的 src。
  20. 添加 width 和 height 属性，以避免页面抖动。
  21. 有下载需求的图片采用 img 标签实现，无下载需求的图片采用 CSS 背景图实现。
  22. 有文本标题的控件必须使用 label 标签将其与其标题相关联。
  23. 使用 button 元素时必须指明 type 属性值。
