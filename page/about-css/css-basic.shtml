易忘记css

  • CSS3word-wrap:break-word;允许长单词或 URL 地址换行到下一行。
  • white-space:pre空白会被浏览器保留 | nowrap文本不会换行
  • CSS3word-break:break-all允许在单词内换行 | keep-all只能在半角空格或连字符处换行;
  • z-index:number
    失效的情况：1、父标签position为relative 2、本身无position属性 3、本身是float属性
  • CSS3text-overflow:clip修剪文本; | ellipsis省略符号来代表被修剪的文本; | string给定的字符串来代表被修剪的文本;
    需要块级元素，配合white-space:nowrap;overflow:hidden;才生效
  • CSS3text-shadow:可向文本应用阴影。能够规定水平阴影、垂直阴影、模糊距离，以及阴影的颜色;
  • CSS3column-count:auto | number规定元素应该被划分的列数;
  • CSS3resize:none无法调整元素的尺寸; | both可调整元素的高度和宽度; | horizontal可调整元素的宽度; | vertical可调整元素的高度;
    此属性生效，需要设置元素的 overflow 属性，值可以是 auto、hidden 或 scroll
  • CSSoutline:outline-color | outline-style | outline-width
    绘制于元素周围的一条线，位于边框边缘的外围
  • CSS3outline-offset:length轮廓与边框边缘的距离;
    对轮廓进行偏移，并在边框边缘进行绘制
