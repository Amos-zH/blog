易忘记css

  • word-wrap:break-word;允许长单词或 URL 地址换行到下一行。
  • white-space:pre空白会被浏览器保留 | nowrap文本不会换行
  • word-break:break-all允许在单词内换行 | keep-all只能在半角空格或连字符处换行;
  • z-index: number
    失效的情况：1、父标签position为relative 2、本身无position属性 3、本身是float属性
  • text-overflow:clip修剪文本 | ellipsis省略符号来代表被修剪的文本 | string给定的字符串来代表被修剪的文本
    需要块级元素，配合white-space:nowrap;overflow:hidden;才生效
