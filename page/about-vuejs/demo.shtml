  1. v-if="age > 25" Age: {{ age }} yes Name: {{ name }}

    可以用v-else指令为v-if添加一个“else块”。v-else或v-else-if元素必须立即跟在v-if元素的后面——否则它不能被识别

  2. v-show

    Name: {{ name }}

    v-show相当于设置display属性，v-show 不支持 template 元素，也不支持 v-else。
